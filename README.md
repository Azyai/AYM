# AYM - AI 面试助手

一款免费的 Windows 桌面端 AI 面试辅助工具，以悬浮透明窗口的形式运行在屏幕上，帮助你在面试过程中实时获取 AI 辅助回答。

> 软件使用完全免费，源代码暂不公开。

---

## 功能特性

### 语音实时问答

自动捕获系统音频或麦克风输入，通过语音识别转文字后发送给大模型，实时流式返回 Markdown 格式的参考答案。支持三种工作模式：

- **手动模式** — 按 `Ctrl+Alt+Space` 开始/停止录音，Push-to-Talk 风格
- **自动侦听** — VAD 语音活动检测，自动识别说话、自动停止、自动生成答案，全程无需手动操作
- **混合模式** — 自动侦听在后台运行，同时支持手动快捷键随时介入

### 截图识题

按 `Ctrl+Shift+Q` 框选屏幕上的面试题目区域，通过视觉大模型直接理解图片内容并给出解答，支持选择题、填空题、编程题、简答题等多种题型。

### 个人知识库（Wiki）

上传你的项目经历、实习经历、工作经历的 Markdown 文档，AI 回答涉及个人经历的问题时，会自动检索你的文档内容，基于你**真实的经历**来生成回答，而非泛泛而谈。

- 支持按章节精准检索（项目概述、核心技术实现、难点与解决方案、量化成果等）
- 高频追问 Q&A 回退机制，面试官深入追问也能覆盖
- A/B 类问题自动分类，通用技术问题直接回答，个人经历问题调取文档

### 四种面试场景

技术面、综合面、HR 面、算法面，各有针对性的 Prompt 模板。可配置个人技术栈和自定义热词，让回答更贴合你的背景，同时提升语音识别的专业术语准确率。

### 隐蔽设计

- **防截屏/防录屏** — 基于 Windows `SetWindowDisplayAffinity` 实现，腾讯会议、钉钉、Zoom 等屏幕共享中完全不可见
- **鼠标穿透** — 点击浮窗会直接穿透到后面的应用，不影响正常操作
- **不抢焦点** — 窗口永远不会获取输入焦点，不触发切屏检测
- **隐式快捷键** — 全局快捷键仅使用键盘左侧修饰键，在系统层面完全不可见

### 局域网共享

开启后同一局域网内的其他设备可通过浏览器实时同步查看问答内容，适合双人协作场景。

### 历史记录导航

每次完整的问答都会保存为一条会话记录，可通过侧边栏或快捷键 `Ctrl+W` / `Ctrl+S` 前后翻阅。

## 快捷键一览

| 快捷键               | 功能              |
| -------------------- | ----------------- |
| `Ctrl + Alt + Space` | 开始/停止语音录制 |
| `Ctrl + Shift + Q`   | 截图识别屏幕题目  |
| `Ctrl + B`           | 展开/收起回答面板 |
| `Ctrl + H`           | 显示/隐藏整个窗口 |
| `Ctrl + W`           | 上一条问答记录    |
| `Ctrl + S`           | 下一条问答记录    |
| `Ctrl + A`           | 回答区域向上滚动  |
| `Ctrl + D`           | 回答区域向下滚动  |
| `Ctrl + Alt + K`     | 退出软件          |

> 所有快捷键仅使用键盘**左侧**的 Ctrl / Alt / Shift，右侧修饰键不受影响。

## 技术栈

| 层级     | 技术                         |
| -------- | ---------------------------- |
| 桌面框架 | Tauri v2                     |
| 前端     | React 19 + TypeScript + Vite |
| 后端     | Rust (Tokio 异步运行时)      |
| 文本回答 | DeepSeek                     |
| 截图识别 | 通义千问 (DashScope)         |
| 语音识别 | 豆包 (火山引擎) / 讯飞       |
| 音频捕获 | cpal (WASAPI Loopback)       |
| 安装打包 | NSIS                         |

所有 AI 服务均使用用户自己的 API Key，软件本身不收取任何费用。

## 使用说明

### 系统要求

- Windows 10 / 11 (x64)
- WebView2 Runtime（通常已预装）

### 安装

1. 前往 [Releases](../../releases) 页面下载最新安装包（`.exe`）
2. 运行安装程序，按提示完成安装
3. 首次启动需输入激活码（联系我获取，免费）

### 配置

启动后点击窗口右上角的设置按钮，进行以下配置：

1. **API Key** — 填入 DeepSeek API Key（必需），通义千问 / 讯飞 / 豆包 Key 按需配置
2. **面试场景** — 选择技术面 / 综合面 / HR 面 / 算法面
3. **技术栈** — 填入你的技术栈关键词（如 Java、Spring Boot、MySQL），会注入到 Prompt 中并提升语音识别准确率
4. **工作模式** — 手动 / 自动侦听 / 混合
5. **个人知识库** — 上传你的项目经历 Markdown 文档（可选但强烈推荐）

## 个人知识库文档格式

Wiki 文档为 Markdown 格式，建议按以下结构组织：

```markdown
---
id: 智能运维助手
type: project
---

# 智能运维助手

## 基本信息
- 时间：2025.09 - 2026.01
- 角色：核心开发者
- 技术栈：Spring Boot / LangChain4j / Milvus / Redis

## 项目概述
（项目背景、目标、整体架构简介）

## 职责与贡献
（你在项目中具体做了什么）

## 核心技术实现
（关键技术方案的详细描述）

## 难点与解决方案
（遇到的挑战和解决思路）

## 量化成果
（可量化的成果数据）

## 高频追问 Q&A
（预设面试官可能的追问及参考答案）
```
